Hamburger Recipe

Description

This is a food that you can never go wrong with, a class hamburger.

Ingredients

  • Bread
  • Beef Patties
  • Cheese

Steps

  1. Start cooking the beef patties
  2. Warm up and toast the bread/buns
  3. Place the beef patty once cooked inside of the bread
  4. Add cheese and other toppings if you want to this new tasty Hamburger
